1. Явный содержит информацию о классе компонента. Неявный не содержит, использует фильтр для определения
2. Когда система получает неявный объект, она выполняет поиск наиболее подходящего по 3 критериям (данные, действия, категория)
3. С помощью онКреэйт
4. Определяет, при каких обстоятельствах выполнится действие
5. ACTION SEND, ACTION INSERT, ACTION VIEW
6. Указывают системе, как запускать активность и обработать это после запуска
7. сбор фильтров - фильтры, не соотв. действию удаляются - сравнение части пути с тегом дата
8. Место хранения личных данных приложения, хранятся и используются этим приложением. Другие приложения не имеют доступа к данным
9. С помощью MODE_PRIVATE, написанный файл будет использоватся приложением, что создало файл
10. Внешнее хранилище, к файлам не применяется система безопасности  
11. SharedPreferences - постоянное хранилище на андроид, используемое приложениями для сохранения своих настроек
